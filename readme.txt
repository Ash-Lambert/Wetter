 Installation:
 
 1. Alle Dateien inklusive Unterordner auf ein beliebiges Laufwerk kopieren.
 
 2. In der Datei index.html in Zeile 8 den Pfad zu jquery angeben.
 
 3. Im Ordner js in der Datei wetter.js den link zu openweathermap.org ändern.
	Sie benötigen einen Account auf dieser Seite!
	Ändern Sie "IhreStadt" und "IhreID" durch die von Ihnen gewünschte Stadt und ihre ID.

 4. Ändern Sie in Zeile 43 "Stadtname" in die von Ihnen ausgewählte Stadt.

 5. In Zeile 54, 58, 62, 66, 70, 74, 78, 82, 86, 90, 94, 98 und 102 den link bzw. Pfad zu einem Bild eingeben.

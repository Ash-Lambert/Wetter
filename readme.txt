 CSS optimiert für die Anzeige auf Smartphones. Die Desktopansicht ist noch nicht optimal.
 
 Installation:
 
 1. Alle Dateien inklusive Unterordner auf ein beliebiges Laufwerk kopieren.
 
 2. In der Datei "index.html" in Zeile 8 den Pfad zu jquery angeben. Am einfachsten runterladen unter:
    https://jquery.com/download/
    (Download the compressed, production jQuery 3.4.1) Rechte Maustaste: "Ziel speichern unter..." und im Projektpfad im Ordner "js" speichern.
    Dann wäre der Pfad, der anzugeben ist: "js/jquery-3.4.1.min.js"
 
 3. Im Ordner "js" in der Datei "wetter.js" den link zu openweathermap.org ändern.
    Sie benötigen einen Account auf dieser Seite um eine ID zu erhalten!
    Ändern Sie in Zeile 7 in dem link die Wörter "IhreStadt" und "IhreID" durch die von Ihnen gewünschte Stadt und ihre ID.

 4. Ändern Sie in der Datei "wetter.js" in Zeile 43 den Text "Ihre Stadt" in die von Ihnen ausgewählte Stadt.

 5. In der Datei "wetter.js" in Zeile 54, 58, 62, 66, 70, 74, 78, 82, 86, 90, 94, 98 und 102 den link bzw. Pfad zu einem Bild eingeben.
    Einfach mal googeln nach "Wetter Icons" oder die Icons von https://openweathermap.org/weather-conditions nehmen.
 
 6. Zack, fertig!
